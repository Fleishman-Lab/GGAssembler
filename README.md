# Dawdlib
Create a cost sensitive degenerate codon sequence

## Requirments
* python >= 3.6

## Install

1. clone the repo: `git clone https://github.com/Fleishman-Lab/dawdlib.git`
2. Install using pip: `pip install dawdlib`
